# Task 1: Sentiment Analysis of X (Twitter) Data

## Project Overview

This project performs sentiment analysis on X (formerly Twitter) data using Natural Language Processing (NLP) and Machine Learning techniques. The objective is to classify tweets into Positive, Neutral, and Negative sentiment categories and derive insights from social media discussions.

## Dataset

* Total Tweets: 162,969
* Sentiment Classes:

  * Positive (1)
  * Neutral (0)
  * Negative (-1)

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn
* WordCloud

## Project Workflow

1. Data Loading and Exploration
2. Data Cleaning and Preprocessing
3. Exploratory Data Analysis (EDA)
4. Sentiment Distribution Analysis
5. Word Cloud Generation
6. TF-IDF Feature Extraction
7. Model Training and Evaluation
8. Performance Comparison

## Machine Learning Models

### Logistic Regression

* Accuracy: 85.57%

### Naive Bayes

* Accuracy: 70.45%

## Results

| Model               | Accuracy |
| ------------------- | -------- |
| Logistic Regression | 85.57%   |
| Naive Bayes         | 70.45%   |

Logistic Regression outperformed Naive Bayes and achieved the highest accuracy for sentiment classification.

## Visualizations

* Sentiment Distribution Chart
* Positive Word Cloud
* Negative Word Cloud
* Top 20 Frequent Words
* Model Accuracy Comparison
* Confusion Matrix

## Conclusion

The project successfully classified tweets into sentiment categories using NLP techniques. Logistic Regression combined with TF-IDF feature extraction provided the best performance with an accuracy of 85.57%.
